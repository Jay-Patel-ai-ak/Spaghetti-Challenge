# Spaghetti-Challenge
It refers to to a Git repository with a messy, intertwined commit history that is difficult to navigate or understand.</br>
It represent the messy state of repository.  

### How to avoid Spaghetti-Challenge ?
##### To Keep repository 'UNTANGLED'(Unorganised, Unstructured); Use following points 
1. Feature Branching 
2. Rebasing 
3. Merge Policies ( Resolving Merge Conflicts)
